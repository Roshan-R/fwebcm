# fwebcm

Play videos as your webcam feed !

## Installing dependencies

### Ubuntu

 ` sudo apt install v4l2loopback-dkms ffmpeg zenity ` 
### Fedora
 ` sudo dnf install akmod-v4l2loopback ffmpeg zenity `
 ## Running
 
 make the script executable and enjoy!
 ```
 chmod +x webcam.sh 
 ./webcam.sh
 ```
